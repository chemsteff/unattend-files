Optimizations:
 1. Disable Windows Defender
 2. Disable System Protection / System Restore
 3. Enable long paths
 4. Enable Remote Desktop services (RDP)
 5. Harden ACLs (Removes write permissions on C:\ for the Authenticated Users group)

 7. Do not update Last Access Time stamp
 8. Do not reboot with users signed in
 9. Turn off system sounds
 10. Disable app suggestions
 11. Disable widgets
 12. Audit process creation events
